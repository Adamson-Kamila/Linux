Создание новой виртуальной машины:

Создать -> в текстовой строке даем имя машине -> в строке Образ ISO выбираем скаченны файл (Ubunto).
-> поставить галочку в "Пропустить автоматическую установку" -> в настройках выбираем: ->
-> основная память (4000 МБ), процессоры (от 2-4 ЦП), жесткий диск (30-35 МБ) -> готово.

Переходим к настройкам (факультативно, кроме сети):

Настроить -> (Общие: вкладка (дополнительно) можно настроить общий буфер обмена, например "из основной в гостевую")->
-> (Аудио (если музыка не нужна, убираем галочки)) -> 
-> (Сеть (сетевой мост, либо Nat)) -> OK

Запускаем виртуальную машину.
На экране выбираем "Try or install ubuntu" -> enter ->
-> выбираем язык English -> Install Ubuntu -> English, English ->
-> убираем галочку с "Download updates while installing Ubuntu" ->
-> installation type  выбираем continue -> в появившемся окне выбираем continue.
Подтверждаем свое местоположение 
Задаем себе имя, создаем пароль. Галочку на "require my password to log in" оставляем -> continue.

Заходим в виртуальную машину. Правой кнопкой мыши на экране выбираем "open in terminal"
Необходимо установить пакеты для работы - mc и ssh
В терминале вводим:sudo apt update -> пароль (в закрытую). Если запросы не пойдут, то сеть настроена некоррректно, необходимо будет переустановить.
Далее вводим - sudo apt install ssh mc -> нажимаем enter

Открываем терминал gitbush, подключаем ssh к удаленной машине.
В терминале ubuntu вводим команду ip a. Если настроен мост, то адрес юудет начинаться со 192.168. далее рандом.
В терминале gitbush подключаем протокол ssh, вводим команду:
-> ssh имя пользователя@ ip adress -> yes -> вводим пароль

Для того чтобы экран ubuntu выводился на полный экран, качаем еще один пакет.
sudo apt install gcc perl make -> yes
Далее, в верхнем меню выбираем "Устройства" -> подключить образ диска дополнений гостевой ОС. В боковом меню выбираем диск -> правой кнопкой мыши по экрану выбираем open in terminal.
В терминае ввыодим команду bash autoran.sh -> вводим пароль. Ждем установки. 
Посе загрузки делаем перегазгрузку системы через терминал - > sudo reboot.
Вводим пароль, в верхнем меню экрана выбираем "Вид" -> подгонять размер экрана гостевой ОС.
